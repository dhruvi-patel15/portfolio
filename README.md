# Dhruvi Patel Portfolio

This is a modern, responsive personal portfolio website built with HTML, CSS, and JavaScript. It showcases my introduction, skills, education, resume, and contact details in a polished, professional layout.

## Features
- Responsive design for desktop and mobile
- Clean hero section with profile image
- About, Skills, Education, Resume, and Contact sections
- Smooth scrolling and animated reveal effects
- Working contact links and resume download

## Tech Stack
- HTML5
- CSS3
- JavaScript

## Project Structure
- INDEX.html - Main page structure
- style.css - Styling and responsive layout
- script.js - Interactivity and animations
- assets/ - Images and resume files
